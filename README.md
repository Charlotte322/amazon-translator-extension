# 亚马逊中英实时互译浏览器插件（本地大模型版）

[English](README-EN.md)

### 项目简介
一款专为亚马逊购物场景深度优化的浏览器插件，**全程调用本地微调大模型**实现中英实时自动互译。核心聚焦亚马逊买家对话框全场景——覆盖客服选项、商品卡片信息、买家与客服的所有对话内容，无需依赖第三方翻译API，隐私更安全、翻译响应更即时，且通过场景化微调实现远超通用翻译工具的精准度。

插件后台直接对接微调后的本地大模型 `Llama3.1-8b-q4_k_m.gguf`，无需联网即可完成翻译，完美适配跨语言购物沟通需求，告别语言障碍导致的订单纠纷、商品误解等问题。

### 核心功能
1. 一键触发翻译：点击插件按钮，即时显示全场景翻译结果，操作零门槛
2. 智能语言检测：自动识别中英文内容，无需手动切换翻译方向，适配混合语言场景
3. 无缝翻译体验：无需刷新页面，翻译过程与浏览、对话操作无冲突，流畅不卡顿
4. 非侵入式设计：翻译弹窗不遮挡页面核心内容，不影响正常购物流程
5. 全场景覆盖：支持买家对话框、客服选项、商品卡片、对话历史等所有目标场景翻译
6. 术语精准匹配：针对亚马逊商品分类、交易术语、客服话术优化，翻译无歧义

### 安装步骤
#### Chrome 浏览器
1. 下载本仓库代码，或从 Releases 页面下载打包好的插件
2. 打开 Chrome，输入 `chrome://extensions/` 进入「扩展程序管理页」
3. 开启页面右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目的 `chrome-extension` 文件夹
5. 浏览器右上角出现插件图标，安装完成
6. 点击插件图标，在弹窗中配置本地 `Llama3.1-8b-q4_k_m.gguf` 模型的文件路径，保存即可

#### Edge 浏览器
1. 下载本仓库代码，或从 Releases 页面下载打包好的插件
2. 打开 Edge，输入 `edge://extensions/` 进入「扩展管理页」
3. 开启页面右上角的「开发者模式」（不同版本位置可能略有差异）
4. 点击「加载已解压的扩展」，选择本项目的 `chrome-extension` 文件夹
5. 浏览器右上角出现插件图标，安装完成
6. 点击插件图标，在弹窗中配置本地 `Llama3.1-8b-q4_k_m.gguf` 模型的文件路径，保存即可

### 使用方法
1. 打开任意亚马逊商品详情页，点击右侧「Sold by」旁边的卖家名称，在「Have a question for [卖家名称]?」下方点击「Ask a question」，即可进入客服沟通页
2. 确认插件已配置好本地大模型路径
3. 点击浏览器右上角插件图标，开启翻译功能
4. 页面中所有目标内容（客服选项、商品卡片、对话文本等）将自动实时翻译，无需额外操作
